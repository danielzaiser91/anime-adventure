# Chronicles of the Celestial Blade — Claude Instructions

## Versionierung (PFLICHT)

**Vor jedem Commit** die Version in `package.json` bumpen:
- Bugfix / Hotfix → Patch: `1.0.0` → `1.0.1`
- Neues Feature → Minor: `1.0.0` → `1.1.0`
- Breaking Change → Major: `1.0.0` → `2.0.0`

Keine Ausnahmen — auch reine Workflow- oder Config-Änderungen bekommen einen Patch-Bump.
